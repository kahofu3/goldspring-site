GitHub Token Settings

Required to save content to the repository

How to generate a GitHub Personal Access Token:
  1. Go to github.com and sign in as kahofu3
  2. Navigate to Settings → Developer Settings
  3. Click Personal Access Tokens → Tokens (classic)
  4. Click Generate new token (classic)
  5. Set a note (e.g. "goldspring-cms") and select the repo scope
  6. Click Generate token and copy it here

The token is stored only in your browser's localStorage — it is never sent anywhere except GitHub's API.
